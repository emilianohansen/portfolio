# Emiliano Hansen Portfolio Website

A GitHub Pages-ready portfolio website with a visual homepage, circular project previews, a minimal research project gallery, individual case-study pages, and a sketchbook page.

## Publish on GitHub Pages

1. Create a new public repository named `emilianohansen22.github.io`.
2. Upload the contents of this folder, not the folder itself.
3. Go to **Settings → Pages**.
4. Set **Source** to **Deploy from a branch**.
5. Set **Branch** to `main` and folder to `/root`.
6. Save and wait for GitHub to publish the site.

## Edit content

- Homepage: `index.html`
- Project gallery: `research.html`
- Sketchbook: `sketchbook.html`
- Individual project pages: `projects/`
- Styles: `assets/styles.css`
- Images: `assets/img/`
