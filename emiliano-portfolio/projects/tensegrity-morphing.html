Compliant Mechanisms

Shape-morphing tensegrity structures

Compliant tensegrity systems using springs, tendons, balloons, and layer-jamming for shape change.

Overview

This project explores tensegrity structures as adaptive robotic bodies that can morph, stiffen, and interact with their environment.

Design approach

The design space includes spring-centered struts, vacuum-based layer jamming, tendon actuation, and soft membrane elements for programmable deformation.
