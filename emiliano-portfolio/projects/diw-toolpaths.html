Digital Fabrication

Rhino/Grasshopper DIW toolpath generation

Custom computational workflows for generating direct-ink-writing paths on complex soft robotic surfaces.

Overview

This project turns CAD geometry into machine-ready toolpaths for printing conductive circuits and patterned features on soft robot substrates.

Design approach

The workflow uses Rhino/Grasshopper to define unwrapped geometry, angled hatch patterns, sorted toolpaths, layered Z steps, and G-code export for DIW experiments.
