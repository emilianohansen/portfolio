Soft Robotics · SDU

Kirigami-based snake-inspired soft robot

Flexible sensor integration and kirigami actuator design for studying snake-like soft robotic locomotion.

Overview

This project explores soft robotic bodies that use patterned compliance and embedded sensing to understand locomotion and body–environment interaction.

Design approach

The system combines kirigami mechanics, silicone-based flexible structures, and direct-ink-written conductive traces to create a sensing layer on top of deformable actuator architectures.
